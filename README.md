# 🌦️ Weather App Web Application

## 📖 Description

The Weather App is a web-based application that allows users to check real-time weather conditions of any city. It fetches live weather data using a weather API and displays important details such as temperature, humidity, and wind speed.

This project demonstrates the integration of APIs with front-end technologies to build dynamic and interactive web applications.

---

## 🎯 Problem Statement

Users often need quick and accurate weather information, but switching between multiple apps or websites can be inconvenient.

This project solves the problem by:

* Providing real-time weather data in one place
* Allowing users to search weather by city
* Displaying essential weather details in a simple UI

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Display temperature
* 💧 Show humidity levels
* 🌬️ Show wind speed
* ❌ Error handling for invalid inputs
* 📱 Responsive design
* ⚡ Real-time API data fetching

## ▶️ Usage

* Open the application
* Enter a city name
* Click search
* View real-time weather data

---

## 🔑 API Setup (IMPORTANT)

This project uses a weather API to fetch data.

Steps:

1. Get API key from OpenWeather website
2. Open `script.js`
3. Replace with your API key:

Example:
const apiKey = "YOUR_API_KEY";

⚠️ Do not expose your API key publicly in production
